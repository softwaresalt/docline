<!-- BEGIN:ImplementationNotes -->
Default change: `read_pdf(path, *, layout_engine: str = "auto")` — this is technically a behavior change at the API surface but `"auto"` resolves to `"heuristic"` when docling is not installed (i.e., always in core CI), so existing callers see identical behavior unless they install the `pdf` extra. Document this in the docstring. The `CountingPictureSink.media_root` is created lazily on first `.emit()` to avoid creating an empty `media/` dir for sources with zero images.
