Resolve external / split-file OpenAPI $ref values so operation docs cross-link to their schema docs. Today the 050-F renderer links only local (#/components/schemas/X) refs; fabric-rest-api-specs splits every service across files (swagger.json referencing ./definitions.json#/definitions/X and ../common/definitions.json#/...), so all operation->schema references render as empty cells. Measured on the real corpus (docs/decisions/2026-07-05-openapi-external-ref-resolution-spike.md): of 4385 refs across 131 specs, 1027 (23%) resolve today, 2450 (56%) are external structural refs that resolve NOWHERE — 2033 operation-layer (op->schema) + 417 cross-file schema->schema. Operation docs are 0% cross-linked (0 of 661). Resolving these lifts referential coverage 23%->79%, structural edges 973->~3423 (3.5x), and connects the operation layer (661 nodes) to the schema layer (1188 nodes) for graphtor. v1 SCOPE: local relative-file refs only, resolved within the workspace root; URL-valued refs DENIED (SSRF); example (x-ms-examples) refs NOT resolved. Reuse safe_workspace_path/PathContainmentError for containment and 051-F fragment mapping for 2.0 targets.

<!-- BEGIN:dod -->
- An external file ref (./definitions.json#/definitions/X or ../common/definitions.json#/...) in an operation resolves to the schema doc that target file produces, emitted as a relative Markdown link and harvested as an operation->schema edge.
- Cross-file schema->schema refs likewise resolve.
- SECURITY (NON-NEGOTIABLE): file refs are resolved via safe_workspace_path and reject ../ / absolute / symlink escapes above the workspace root (PathContainmentError); URL-valued refs (http/https) are denied, never fetched; a test asserts both.
- Swagger 2.0 external targets get the 2.0->3.x fragment mapping (#/definitions/X -> #/components/schemas/X).
- Example (x-ms-examples) refs are NOT resolved/rendered.
- Cross-file ref cycles terminate (guard across files).
- Runtime: re-ingest fabric-rest-api-specs and confirm operation docs go from 0% cross-linked to ~100%; report the edge delta.
- Quality gates pass (ruff, pyright, pytest, format).
<!-- END:dod -->
