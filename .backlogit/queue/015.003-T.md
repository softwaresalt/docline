Implement `read_docx_blocks_with_media(path, picture_sink) -> tuple[list[str], list[MediaReference]]` in `src/docline/readers/docx.py`. Walk `<w:drawing>` elements inside paragraphs; for each `<a:blip r:embed="rIdN"/>`, look up the `rIdN` relationship in `word/_rels/document.xml.rels` (parsed via `defusedxml`); resolve `Target` to `word/{Target}` (typically `word/media/imageN.{ext}`); read bytes from the zip; pass to `picture_sink.emit(mime, data)`; insert `![](media/{filename})` into the paragraph's emitted markdown at the source position. Skip unresolvable embed IDs silently with a log warning. Defend against path traversal in `Target`. All 7 DOCX image-extraction tests pass. Back-compat path: `read_docx_blocks` unchanged (delegates internally to the new helper with `picture_sink=None`).
