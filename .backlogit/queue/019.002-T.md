- `coalesce_ranges(flagged_indices, *, total_pages, buffer=1, merge_gap=2)` signature with kwargs-only after the first arg.
- Returns sorted list of `(start_inclusive, end_inclusive)` tuples clamped to `[0, total_pages-1]`.
- Input validation: raises `ValueError` for `buffer < 0`, `merge_gap < 0`, or `total_pages < 0`.
- RED tests written first and fail; GREEN after implementation.

Files:

No new dependencies; uses only stdlib.
