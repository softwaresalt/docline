Rewrite cross-doc markdown links of the form [text](other-doc.md) and [text](../subdir/other-doc.md#anchor) so downstream graph extraction can treat them as first-class edges. Two-output strategy: (1) inline link text rewritten to point at the canonical output path (e.g., relative-to-job-root posix path) so the rendered link works in any preview; (2) collected list of typed link relationships in docline:cross_doc_links frontmatter namespace: list of {target_path, anchor, link_text} dicts. External (https://) links and asset (./media/foo.png) links pass through unchanged. Anchor-only links (#fragment) pass through unchanged. Resolution scope: limited to the source corpus being processed; targets outside the corpus emit unresolved {target_unresolved: 'path/that/was/given.md'} entries.
