Make docling OCR conditional so RapidOCR only runs on image-only/scanned pages, not on native-text pages that already carry an extractable text layer. Implements Option A from deliberation 002-DL: thread an explicit do_ocr decision from the triage/batch layer (derived from the text-layer signal the triage pass already computes) down through the docling_worker dispatch into PdfPipelineOptions. Conservative per-range rule: do_ocr=False only when every page in a coalesced range has a healthy text layer; any image-only page forces do_ocr=True. Backward-compatible default (do_ocr=True) when no decision is supplied. Expected to remove OCR cost from native-text corpora (the dominant driver of the ~247min cosmos baseline) without regressing scanned-PDF fidelity. Evidence: docs/decisions/2026-06-22-docling-worker-subprocess-confirmation.md; deliberation 002-DL.
