In pdf_triage/pdf_batch, derive a per-coalesced-range do_ocr decision from the per-page text-layer signal the triage Pass-1 already computes, and set it on the worker manifest/dispatch. Conservative rule: do_ocr=False only when every page in the range has a healthy text layer; any image-only page forces do_ocr=True. Single domain: src/docline/process/pdf_triage.py + pdf_batch.py.
