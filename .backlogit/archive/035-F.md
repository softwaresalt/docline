Fix the misleading per-page docling metric surfaced by the 2026-06-25 cosmos routing investigation. The pa3 engine_distribution counts docling-collapsed placeholder pages (96.9% of 'docling' pages on the cosmos run hold 0 chars) as docling pages, overstating per-page coverage (82% reported, but really 86 ranges -> 86 concatenated blobs). Add range-level docling attribution to the pa3-summary.json so runtime-verification numbers are not misread: report the number of docling ranges (= blobs), content-bearing docling pages, collapsed-placeholder pages, and total docling chars. Keep engine_distribution for backward-compat. Single domain: scripts/pa3_triage_cosmos.py reporting. Evidence: docs/decisions/2026-06-25-cosmos-docling-routing-investigation.md.
