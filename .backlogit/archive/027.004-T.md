Empirical quality + cost study: run ADI against the 5 cosmos sample ranges already in .elt/output/cosmos-triage-022/study/results/ (the same fixtures used for the markitdown-vs-docling extraction-strategy study).

For each sample range:
1. Run ADI via the new pdf_engine=azure_di path
2. Compute the same 25 AST-aware metrics from scripts/study/evaluate_markdown.py
3. Compare ADI output to existing docling splices
4. Record wall time + ADI billable page count

Aggregate metrics:
- ADI vs docling: structural density per 1k chars (target: within 10%)
- ADI vs docling: heading count, table count, list item count (target: within 10%)
- ADI wall time vs docling wall time (target: ADI <5% of docling)
- Total ADI cost (pages * $0.0015)
- Per-page failure count (target: 0)

Output: scripts/study/adi_comparison.py (study harness; reuses build_comparison_dataset.py utilities). Results go to .elt/output/cosmos-triage-022/study/results/adi-findings.json + adi-findings.md.

Cost estimate: 5 ranges x ~50 pages avg = 250 pages = ~$0.40 in ADI fees.
