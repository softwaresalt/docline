title: Derive canonical Learn URL from publish config + source path (v1)
updated_at: 2026-07-04T11:03:39.1885834-07:00

Add a pure `derive_canonical_url(publish_config, source_rel_path) -> str | None` that maps a repo-relative source file path to its canonical Microsoft Learn URL path using the repo's `.openpublishing.publish.config.json`. For a doc under a docset's `build_source_folder`, the URL is `/{url_path_prefix}/{path-under-build_source_folder}` with `.md` dropped, `index.md` collapsed to its directory, forward-slash, host-less, lowercased. Return None when the path matches no docset. v1 scope: base path + url_path_prefix + file→URL normalization. DEFER (documented): monikers, redirect_map, documentId path-depot mappings, docfx.json overrides. Net-new — no openpublishing/docfx reading exists in the codebase today.
